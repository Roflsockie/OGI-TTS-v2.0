⚙️ OGI TTS Application Features

OGI TTS is a multifunctional tool for working with text and speech, supporting various languages and formats.

📋 Main Tabs and Their Functions

🎯 General (Main)

Functions:

  • 🎤 Text-to-Speech: Converting text to speech
  • 📂 Import Text: Loading text from files (.txt, .docx)
  • ▶️ Play Audio: Listening to generated speech
  • 💾 Save Audio: Saving audio to tts_audio folder
  • 🌐 Translation: Translating text between languages
📍 Saving: Audio files are saved to tts_audio/ folder with names like ru_output_edgezira.wav
📊 Batch Processing

Functions:

  • 📁 Import Multiple Files: Loading multiple files at once
  • 🔄 Batch Conversion: Batch text-to-speech conversion
  • 📋 Queue Management: Task queue management
  • 📈 Progress Tracking: Processing progress tracking
📍 Saving: Each file is saved separately to tts_audio/ folder with unique names
🖼️ IMG to Text (Image to Text)

Functions:

  • 📷 OCR Processing: Text recognition in images
  • 🔧 API Integration: Support for Google Vision and Azure Computer Vision
  • 📄 Multiple Formats: Support for PNG, JPG, JPEG, BMP, TIFF, WebP
  • 🌍 Multi-language: Text recognition in different languages
📍 Saving: Recognized text can be copied or used for further processing
⚙️ Settings

Functions:

  • 🎨 Theme Selection: Interface theme selection (Default, OgiDark)
  • 🌐 Language: Interface language selection (English, Русский, Українська)
  • 🔑 API Keys: Setting up keys for translations and OCR
  • 🎵 Voice Settings: Voice parameters setup (speed, volume, pitch)
📋 Log

Functions:

  • 📝 Operation Log: Program operations journal
  • 📋 Copy Log: Copying journal to clipboard
  • 🗑️ Clear Log: Clearing journal
  • 🎨 Color Coding: Color indication of message types
❓ Help

Functions:

  • 📚 API Guides: Detailed API setup instructions
  • 🔗 External Links: Links to provider documentation
  • 💡 Tips & Tricks: Usage tips
  • 🚀 Feature Overview: Description of all program features

📁 File Operations

📥 Document Import

  • Supported formats: .txt, .docx
  • Encoding: UTF-8 (recommended)
  • Maximum size: Limited by system memory
  • Batch processing: Up to 100+ files simultaneously

📤 Audio Export

  • Format: WAV (high quality)
  • Save folder: tts_audio/
  • Naming: language_output_modelvoice.wav
  • Examples: ru_output_edgezira.wav, en_output_edgearia.wav

🎵 Voice Settings

Speech synthesis parameters

  • Speed: 0.5x - 2.0x (controlled by slider)
  • Volume: 0% - 100% (controlled by slider)
  • Pitch: -50Hz - +50Hz (controlled by slider)
  • Voices: Male/Female for each language
  • Languages: English, Russian, Ukrainian, Japanese

💻 System Requirements

Minimum requirements

  • OS: Windows 10/11
  • Python: 3.8+
  • Memory: 4GB RAM
  • Disk space: 500MB
  • Internet: For API functions

🚀 Quick Start

  1. Install dependencies: pip install -r requirements.txt
  2. Run the program: python main.py
  3. Select language and voice in dropdown lists
  4. Enter text or import file
  5. Click "Play" to listen or "Save" to save
